- Run the notebook setup_subscription ( first part ) to get a token for you assistant | don't run the subscription yet.
- Go to api/get_assistant_token.py and setup the token there. ( token only lasts 1 hour )
- Expose your localhost any tool we are using ( localtunnel ) link: https://theboroer.github.io/localtunnel-www/
- Run the project using uvicorn main:app --reload
- [Optional] You can check if you have any old subscriptions and delete them if you want.
- Run the notebook setup_subscription ( second part ) to subscribe your application in Microsoft API Subscription.
- You are good to go now chat with your assistant on Teams.
- Happy chatting !
